
# SHA 1 produces 20 bytes

Each byte represents a different part of an image, for example a dwarf might have:

1. Face colour
2. Beard colour
3. Moustache colour
4. Hair colour
5. Eye colour
6. Clothing colour
7. Face shape
8. Front Hair shape
9. Beard shape
10. Moustache shape
11. Ear shapes
12. Eye shapes
13. Eyebrow shapes
14. Nose shape
15. Mouth shape
16. Eyebrow colour
17. Back Hair shape
18. Clothing style
19. Scars
20. Features
